# ページ用写真の配置

各ページで表示する写真は、このフォルダ以下に配置し、`lib/page-images.ts` でパスを指定してください。

## 推奨ファイル名と配置場所

| ページ | フォルダ | 推奨ファイル名 | 表示位置 |
|--------|----------|----------------|----------|
| ホーム | `home/` | `hero.jpg` | ヒーロー右側 |
| 会社概要 | `about/` | `hero.jpg` | タイトル下 |
| お問い合わせ | `contact/` | `hero.jpg` | タイトル下 |
| お知らせ | `notice/` | `hero.jpg` | タイトル下 |
| 企業向け | `company/` | `hero.jpg` | タイトル下 |
| 学生向け | `student/` | `hero.jpg` | タイトル下 |
| メンバー | `members/` | `hero.jpg` | タイトル下 |
| プライバシーポリシー | `privacy/` | `hero.jpg` | タイトル下 |

## 有効にする手順

1. 上記のフォルダに写真を配置（例: `public/images/home/hero.jpg`）
2. `lib/page-images.ts` を開き、該当するプロパティにパスを設定  
   例: `homeHero: "/images/home/hero.jpg"`

写真を設定しない場合は、該当プロパティを `null` のままにしておくと、画像エリアは表示されません（ホームは1カラム、他ページはタイトルのみ）。
